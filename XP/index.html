Experience Points

In theory, XP for gold is a good idea. In practice, it can be a bit tedious.

Instead of using XP for gold, as Moldvay intended, we're going to use the (slow) XP progression of Worlds Without Number; and, time and interest permitting, we'll use Dungeon World / Blades in the Dark-style questions to award that XP. Players will be able to check the questions they want to be awarded XP for at the end of each session; and the answers to these questions will signal to the GM what the players want to do in the next / coming sessions.

Default WWN XP rewards

The default WWN XP system gives 3 XP "for each game session in which they tried to do something adventurous or in pursuit of someone's character goals," even if their efforts ended in failure. This approach "encourages players to go out and adventure... so long as they are trying to do something worthwhile."[^ Worlds Without Number, p. 255]

Question-based XP

The idea here is that the default questions can help facilitate and encourage the kind of play you want to see at the table.

Note that this system might mean that character progression is slower than the default WWN system.
